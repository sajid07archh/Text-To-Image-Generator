# Text-To-Image-Generator

Project Status: Under Development

This project is a Text-to-Image Generator that converts textual prompts into images using an AI model.
Currently, the project uses a basic model implementation for generating images. Future updates will include improved models, better image quality, and additional features.

## Features

- Generate images from text prompts
- Simple and beginner-friendly implementation
- Can generate multiple images
- Stable Diffusion and Hugging Face is used for this project
- Easy to run and modify
- UI ineterface


## Tech Stack

- Python
- Transformer , Gradio, PyTorch, Hugging Face, Diffusers
- Google Colab
- API Integration

## How It Works

1. User enters a text prompt
2. The AI model processes the text
3. The model generates an image based on the prompt
